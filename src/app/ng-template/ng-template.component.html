NG TEMPLATE

When should ng-template be used???

Options 1:

Used in conjunction with Angular's Structure Directives, such as *ngIf

=> For example:

Current age: {{ user.age }}

=> Bạn đủ tuổi để có thể xem nội dung PG-13

=> Bạn không đủ tuổi để có thể xem nội dung PG-13

Options 2:

When several UI elements in a component are duplicated within the component itself, but that piece of code is too small to separate as a separate component.

=> For example:

You have selected .
There are was selected.
You have selected .
{{ counter }} items

Options 3:

Use ng-template to pass another component. Template override support is available in the component.

=> For example:

{{ l }}
